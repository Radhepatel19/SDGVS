# 🌐 Smart Digital Gram Vikas System (SDGVS)

Smart Digital Gram Vikas System (SDGVS) is a powerful digital platform designed to connect villagers with the Gram Panchayat and modernize rural governance. The application improves transparency, communication, and accessibility of essential services, helping build smart and digitally empowered village communities.

---

## 📌 Overview

SDGVS is built to bridge the gap between rural citizens and local administration. It enables real-time communication, complaint management, access to government schemes, and agricultural support — all in one platform.

---

## 🚀 Key Features

- 📢 **Digital Notice Board** – Official announcements from Panchayat  
- 🧾 **Complaint Management** – Register and track issues  
- 📂 **Document Locker** – Store and manage important documents  
- 🌾 **Farmer Advisory** – Weather alerts, crop guidance, mandi prices  
- 🗳️ **Community Polls** – Participate in village decisions  
- 🚨 **Emergency Services** – Quick access to important contacts  
- 📜 **Government Schemes** – Explore and apply for schemes  
- 🔔 **Notifications** – Real-time updates and alerts  

---

## 🔄 User Flow (Simplified)

1. User selects language (English / Hindi / Gujarati)  
2. Login using mobile number with OTP verification  
3. Complete profile setup (Name, Village, Photo)  
4. Access dashboard with all services  
5. Perform actions (complaints, schemes, notices, etc.)  
6. Receive updates via notifications

## 🧑‍💻 Tech Stack

- **Mobile App:** Flutter  
- **Admin Panel:** MERN Stack (MongoDB, Express.js, React.js, Node.js)  
- **Backend API:** REST API  
- **Database:** PostgreSQL  
- **Cloud Storage:** Cloudinary
